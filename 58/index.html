Обзор

DRT Framework — это комплексный фреймворк для DDoS-атак и бомбинга с визуальным модулем и оптимизатором.

Требования

  • Python 3.8 или выше
  • pip
  • Git (опционально)

Структура проекта

drt/
├── attacks/          # Методы DDoS-атак
├── bomber/           # Сервисы бомбинга
├── core/             # Основная функциональность
├── optimizer/        # Оптимизация производительности
├── visual/           # Визуальный модуль
├── config/           # Конфигурационные файлы
├── utils/            # Вспомогательные функции
└── main.py           # Точка входа

Установка

Пошаговая инструкция по установке

Системные требования

  • Python 3.8+
  • pip 20.0+
  • 4GB RAM
  • 1GB свободного места

Установка из GitHub

git clone https://github.com/yourusername/drt.git
cd drt
pip install -r requirements.txt

Установка зависимостей

# Основные зависимости
pip install rich requests scapy phonenumbers

# Дополнительные зависимости
pip install psutil netifaces pyfiglet

# Для разработки
pip install pytest pytest-cov pytest-xdist

Проверка установки

python -c "import drt; print(drt.__version__)"

DDoS-атаки

9 мощных методов атак

Доступные методы

Метод Описание Протокол
TCP TCP flood атака TCP
UDP UDP flood атака UDP
HTTP HTTP GET flood TCP
HTTPS HTTPS GET flood TLS
SYN SYN flood атака TCP
ACK ACK flood атака TCP
ICMP ICMP echo flood ICMP
DNS DNS amplification UDP
Slowloris Медленная HTTP атака TCP

FAQ

Часто задаваемые вопросы

Что такое DRT Framework?

DRT Framework — это комплексный фреймворк для DDoS-атак и бомбинга с визуальным модулем и оптимизатором.

Какие методы атак поддерживаются?

TCP, UDP, HTTP, HTTPS, SYN, ACK, ICMP, DNS, Slowloris.

Какие сервисы бомбинга доступны?

SMS, Call, WhatsApp, Telegram, Viber, Instagram, Facebook, TikTok, Twitter, VK, Discord и другие.

Есть ли визуальный модуль?

Да, дашборды в реальном времени, графики, анимации и метрики производительности.

Как оптимизировать производительность?

Используйте модуль оптимизатора для оптимизации потоков, сети, пакетов, пропускной способности и задержки.

Кроссплатформенный ли фреймворк?

Да, работает на Windows, macOS и Linux.

Есть ли документация на русском?

Да, вся документация представлена на русском языке.

Как запустить тесты?

cd tests
python run_tests.py --unit
python run_tests.py --integration
python run_tests.py --coverage
