О проекте

ARES Bot — это мощный OSINT‑инструмент с открытым исходным кодом для сбора и анализа цифровых данных. Работает в Telegram, поддерживает Mini App и предлагает гибкие интеграции.

⚡

Быстрый поиск

IP, email, телефон, домен, ник — результат за секунды.

🔒

Приватность

Опциональная маршрутизация через Tor и прокси.

🧩

Модульность

Более 10 независимых источников данных.

Возможности

🌍

Статистика использования

API

Mini App эндпоинт для внешних запросов.

POST /mini-app
Принимает JSON с полем target
{
  "status": "processing",
  "target": "user@example.com"
}

Команды Telegram

/dox цель

Полное досье.

/ip 1.1.1.1

Данные об IP.

/email x@x.com

Проверка утечек.

/phone +7900...

Информация о номере.

/domain example.com

WHOIS, поддомены.

/user nickname

Поиск по соцсетям.

Установка

git clone https://github.com/ares-bot/ares-bot
cd ares-bot
pip install -r requirements.txt
python bot/bot_main.py

Интеграции

  • ip-api.com — геолокация
  • AbuseIPDB — репутация IP
  • Have I Been Pwned — утечки почты
  • NumVerify — валидация телефона
  • WhoisXML — WHOIS
  • SecurityTrails — поддомены
  • Namechk — соцсети
  • GitHub API
