API Конфигурация

REST API для удаленного управления и интеграции

Настройка API

# config/settings.py
API_ENABLED = True
API_HOST = "0.0.0.0"
API_PORT = 8080
API_SECRET = "your-secret-key"
API_TOKEN_EXPIRE = 3600

Аутентификация

POST /api/auth/login
Получение JWT токена
curl -X POST http://localhost:8080/api/auth/login \\
-H "Content-Type: application/json" \\
-d '{"username":"admin","password":"admin123"}'

Запросы с токеном

GET /api/scan
Запуск сканирования
curl -X GET http://localhost:8080/api/scan \\
-H "Authorization: Bearer YOUR_TOKEN" \\
-H "Content-Type: application/json" \\
-d '{"target":"192.168.1.1","ports":[22,80,443]}'

Доступные эндпоинты

Установка

Быстрая установка и запуск DllNire

Требования

Python 3.8 или выше, pip

git clone https://github.com/dllnire/dllnire.git
cd dllnire
pip install -r requirements.txt
python main.py

Или одной командой:

pip install -r requirements.txt && python main.py

Покрытие

Быстрый старт

Пошаговая инструкция для новичков

  1. Клонируйте репозиторий

    git clone https://github.com/dllnire/dllnire.git

  2. Установите зависимости

    pip install -r requirements.txt

  3. Запустите фреймворк

    python main.py

  4. Выберите инструмент

    Введите номер нужного инструмента и следуйте инструкциям

  5. Анализируйте результаты

    Все результаты сохраняются в базу данных и логи
