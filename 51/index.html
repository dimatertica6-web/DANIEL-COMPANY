Установка и запуск

Полное руководство по установке, настройке и первому запуску PornHab. Следуйте инструкциям для быстрого старта.

Системные требования

  • Python 3.8+ — основная среда выполнения
  • pip — менеджер пакетов Python
  • Доступ к интернету — для установки библиотек и работы API
  • Операционная система: Windows 10+, Linux (Ubuntu 20.04+), macOS 11+
  • Оперативная память: от 512 MB (рекомендуется 1+ GB)
  • Свободное место: от 200 MB для кода и баз данных

Для работы с архивами дополнительно требуются утилиты: unzip, unrar, 7z (устанавливаются автоматически на Linux/macOS).

Установка всех библиотек

Скопируйте и выполните команду в терминале для установки всех зависимостей:

pip install requests beautifulsoup4 lxml dnspython phonenumbers psutil python-nmap pyautogui opencv-python numpy pyaudio keyboard selenium cryptography paramiko ftplib scapy tweepy instaloader facebook-scraper python-telegram-bot discord.py slack-sdk boto3 redis pymongo mysql-connector-python psycopg2-binary whois rarfile py7zr pycryptodome pysocks

📌 Примечание: Для работы с архивами также рекомендуется установить:

pip install rarfile py7zr

Общее количество устанавливаемых библиотек — 45+. Время установки может занять 2-5 минут в зависимости от скорости интернета.

Запуск программы

1
Скачайте файл pornhab.py из репозитория или архива
2
Откройте терминал (командную строку) в папке с файлом
3
Выполните команду запуска:
python pornhab.py
4
Введите логин и пароль. По умолчанию: admin / admin. Или зарегистрируйтесь через меню 2.
5
После авторизации откроется главное меню с 150+ функциями. Выбирайте номер и следуйте инструкциям.

💡 Совет: Для удобства создайте ярлык или alias: alias pornhab='python /path/to/pornhab.py'

Структура проекта и файлов

pornhab.py # Основной исполняемый файл users.db # База данных пользователей (SQLite) pornhab_main.db # Основная БД с данными (SQLite) data/ ├── output/ # Экспортированные файлы (JSON, HTML, CSV) ├── logs/ # Логи работы программы ├── screenshots/ # Снимки экрана и веб-камеры ├── extracted/ # Извлеченные архивы ├── scans/ # Результаты сканирования (Nmap, Shodan) └── reports/ # Сгенерированные отчеты в HTML

📁 Все данные сохраняются локально. Базы данных можно просматривать через SQLite Browser или встроенные функции программы (83, 139, 140).

Первые шаги после запуска

  • Шаг 1: Зарегистрируйте пользователя (меню 2) или войдите (меню 1)
  • Шаг 2: Настройте API ключи (меню 143) — для работы функций требуется минимум Telegram или Hunter
  • Шаг 3: Протестируйте базовые функции: 3 (Email Verify), 8 (Port Scanner), 38 (IP Geo)
  • Шаг 4: Для работы с архивами используйте функции 136-142
  • Шаг 5: Экспортируйте результаты через меню 147-149

Получение данных из Telegram — сообщения, пользователи, чаты.

2. Выберите имя и username бота, получите токен вида 123456:ABC-DEF1234ghIkl

Как вставить API ключи в программе

1. Запустите программу и войдите в систему

2. Выберите пункт меню 143 (API Config)

3. Введите название ключа (например: telegram, shodan, hunter)

4. Введите значение ключа (токен или другие данные)

5. Ключ сохранится в памяти программы и будет доступен для всех функций

6. Для проверки работы используйте соответствующую функцию (например, 3 для Email Verify)

Советы по использованию

  • Для быстрых проверок используйте функции 3 (Email), 8 (Ports), 38 (IP Geo) — они не требуют сложной настройки.
  • Для глубокого анализа используйте функции 68 (Nmap), 4 (Shodan), 28 (SecurityTrails) — требуют API ключей.
  • Для работы с архивами используйте функции 136-142 — они автоматически обработают все файлы в текущей папке.

  • Для настройки API используйте функцию 143 — это ускорит работу с внешними сервисами.

База данных — Полная структура

PornHab использует SQLite для хранения всех данных: пользователей, архивов, извлеченной информации и результатов поиска. Все данные хранятся локально.

users.db — Пользователи

Таблица: users - id INTEGER PRIMARY KEY - username TEXT — уникальное имя - password TEXT — SHA256 хеш - created TEXT — дата регистрации - last_login TEXT — последний вход Индексы: - UNIQUE(username) для предотвращения дубликатов

🔐 Пароли хранятся в зашифрованном виде (SHA256). Никакой другой информации о пользователях не сохраняется.

pornhab_main.db — Основная БД

Таблица: archive_index - id INTEGER PRIMARY KEY - filename TEXT - path TEXT - size INTEGER - hash TEXT - extracted TEXT - scanned TEXT - created TEXT Таблица: file_entries - id INTEGER PRIMARY KEY - archive_id INTEGER - filename TEXT - path TEXT - size INTEGER - type TEXT - hash TEXT - content TEXT - created TEXT Таблица: extracted_data - id INTEGER PRIMARY KEY - source TEXT - data_type TEXT - data TEXT - found_in TEXT - timestamp TEXT Таблица: search_results - id INTEGER PRIMARY KEY - query TEXT - result TEXT - source TEXT - timestamp TEXT

Как просматривать и управлять БД

1
Откройте терминал и перейдите в папку с программой
2
Выполните команду для открытия БД:
sqlite3 pornhab_main.db
3
Просмотрите таблицы:
.tables # archive_index file_entries extracted_data search_results
4
Выполните запросы:
SELECT * FROM extracted_data LIMIT 10; SELECT data_type, COUNT(*) FROM extracted_data GROUP BY data_type; .quit

📊 Для удобного просмотра рекомендуется использовать SQLite Browser (бесплатное приложение).

Резервное копирование

Рекомендуется регулярно создавать резервные копии баз данных:

cp pornhab_main.db pornhab_main_backup_$(date +%Y%m%d).db cp users.db users_backup_$(date +%Y%m%d).db

💾 Все данные сохраняются локально. Рекомендуется делать бэкапы перед обновлением программы.

Часто задаваемые вопросы (FAQ)

Ответы на самые частые вопросы о PornHab. Если вы не нашли ответ — свяжитесь с нами.

❓ Какие форматы архивов поддерживаются?

Поддерживаются: ZIP, RAR, 7z, TAR, GZ, BZ2, XZ, TGZ, TBZ2, TXZ, ZST, LZ4, LZMA, LZO, Z, Z, TAZ, TZ, TLZ, TXZ — всего 20+ форматов.

❓ Нужно ли устанавливать дополнительные программы?

Для работы с RAR и 7z архивами требуется установить unrar и 7z. На Linux/macOS это можно сделать через менеджер пакетов. На Windows они обычно устанавливаются автоматически.

❓ Как часто обновляется программа?

Обновления выходят каждые 2-4 недели. Включают новые функции, исправления багов и обновление API. Следите за анонсами в Telegram.

❓ Можно ли использовать PornHab на сервере?

Да, программа работает на любой ОС с Python 3.8+. Рекомендуется использовать screen или tmux для фонового запуска.

❓ Что делать, если функция не работает?

  • Проверьте, установлены ли все библиотеки (команда для установки выше)
  • Проверьте API ключи (меню 143) — многие функции требуют ключей
  • Проверьте интернет-соединение
  • Попробуйте перезапустить программу
  • Свяжитесь с нами в Telegram, если проблема не решается

❓ Безопасно ли использовать PornHab?

Да, программа работает локально и не передает данные третьим лицам. Все API запросы выполняются с вашего устройства. Рекомендуется использовать VPN для дополнительной анонимности.

⚠️ Важно: Используйте программу только в законных целях и с согласия владельцев данных.
