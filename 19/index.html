О проекте

Kali Terminal Ultimate Edition — это не просто эмулятор командной строки, а полноценная экосистема для профессионалов кибербезопасности, DevOps-инженеров и разработчиков. Проект зародился как открытая альтернатива проприетарным терминалам с упором на приватность, скорость и расширяемость. В основе лежит модульное ядро на Python 3, поддерживающее динамическую загрузку плагинов и пользовательских сценариев.

Архитектура терминала позволяет работать с встроенными скриптами на Lua и Python, автоматизировать рутинные задачи с помощью макросов и создавать собственные команды через простой API. Более 200 встроенных утилит покрывают потребности от базовой навигации до криптографических операций, сетевого сканирования и форензики.

  • 200+ команд: файловая система, сеть, криптография, анализ логов
  • Zero telemetry — абсолютное отсутствие сбора данных, телеметрии и аналитики
  • Нативная поддержка Windows (PowerShell + CMD), Linux, macOS и WSL2
  • Модульная система: плагины, кастомные темы и расширения
  • Открытый исходный код (MIT License) — полный аудит безопасности
  • История команд с поиском, экспорт сессий, сниффинг команд в реальном времени
Лицензионная чистота: Все компоненты распространяются под свободными лицензиями. Нет проприетарных библиотек, телеметрии или «звонков домой».
Командная база
ls -lacd ~grep -r ping -c4curl -Ineofetch matrixhash md5portscan topencrypt aes256zip -r cowsayfiglettraceroute netstat -tulpnnslookupdig ssh-keygenbase64hexdump
Расширенная документация: Для каждой команды доступна страница справки (help <command>) с примерами, флагами и практическими кейсами. Встроенный учебник содержит 40+ интерактивных уроков по работе с сетью, шифрованию и автоматизации.
Примеры использования:
portscan 192.168.1.1 -p 1-1024 --threads 50 — быстрое сканирование портов.
encrypt file.txt --algo chacha20 --key ~/.key — шифрование файла.
watch -n 2 "netstat -an | grep ESTABLISHED" — мониторинг соединений.
Подробная установка и зависимости 
# Установка через wget / curl (рекомендовано)
wget -O kali-terminal.zip "https://disk.yandex.ru/d/SQuCnFtV0ZUHHA"
unzip kali-terminal.zip -d kali-terminal
cd kali-terminal
chmod +x install.sh
./install.sh --prefix /opt/kali-terminal

# Ручной запуск (Python 3.7+)
python3 kali-terminal.py --setup-env
python3 kali-terminal.py --theme dark_matrix

# Добавление в PATH (Linux/macOS)
echo 'export PATH=$PATH:/opt/kali-terminal/bin' >> ~/.bashrc

Системные требования: Python 3.7+, pip, git (опционально). Автоматическая установка зависимостей через requirements.txt (colorama, cryptography, psutil, requests).

Для Windows: Используйте PowerShell с правами администратора, предварительно установив Python. Доступен портативный режим без установки — просто запустите kali-terminal.exe из архива.

Пакеты для Arch / Debian: AUR-репозиторий kali-terminal-ultimate-bin и .deb пакет в разделе релизов GitHub.

Политика конфиденциальности и обработка данных

1. Общие положения. Kali Terminal Ultimate Edition (далее — «Программа») разработана с соблюдением принципа Privacy by Design. Разработчик гарантирует, что ни одна версия программы не содержит средств сбора телеметрии, краш-репортов, аналитических модулей, трекеров или рекламных фреймворков. Программа не устанавливает фоновых служб, не открывает скрытых сетевых соединений и не передает данные о пользователе третьим лицам.

2. Локальное хранение. Все пользовательские данные (история команд, настройки, конфигурационные файлы, сессии) сохраняются исключительно в локальных директориях: ~/.config/kali-terminal/ и ~/.local/share/kali-terminal/ (Linux/macOS) или %APPDATA%\KaliTerminal\ (Windows). Никакая информация не синхронизируется с внешними серверами.

3. Сетевые запросы. Программа осуществляет сетевые вызовы только по явной команде пользователя: curl, wget, ping, portscan и аналогичные утилиты. Встроенный модуль обновлений работает по запросу (kali-terminal --update) через GitHub API без передачи идентификаторов устройства.

4. Безопасность шифрования. Команды encrypt / decrypt используют алгоритмы AES-256-GCM и ChaCha20-Poly1305. Ключи хранятся локально. Криптографические операции не используют внешние облачные сервисы.

Юридическая оговорка: Программа распространяется «как есть» (AS IS). Использование сетевых утилит для сканирования чужих систем без разрешения может нарушать законодательство. Пользователь несет полную ответственность.
Технические гарантии приватности
  • Нет телеметрии, краш-репортов, аналитики
  • История команд только локально
  • Шифрование конфигурации мастер-паролем
  • Полностью открытый код — независимый аудит
  • Отсутствие фоновых обновлений и «звонков домой»
  • Поддержка анонимных прокси (Tor, SOCKS5)
Политика конфиденциальности (кратко):
Никакие данные не покидают ваше устройство. Программа не содержит трекеров, не требует регистрации.
