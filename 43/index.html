Установка

NifyDox работает на Python 3.8+. Для установки зависимостей выполните:

python3 NifyDox.py --install

Скрипт автоматически установит pystyle и cryptography. После установки вы можете запускать программу без флага.

  • Кроссплатформенность
  • Авто-инсталлер
  • Минимальные зависимости
  • Работает в любой ОС

Если у вас уже установлены зависимости, пропустите флаг --install и переходите к запуску.

Запуск

После установки достаточно выполнить в терминале:

python3 NifyDox.py

При первом запуске будет запрошен пароль. По умолчанию: Dox (регистр важен). После авторизации откроется главное меню со всеми доступными командами.

Для завершения работы используйте /exit или Ctrl+C. Все данные хранятся в зашифрованном виде.

Основные команды

Все команды начинаются с /. Примеры:

/phone +71234567890   /ip 192.168.0.1   /email user@domain.com
/name Иванов Иван   /photo img.jpg   /ddos 1.1.1.1:80
  • /phone — поиск по номеру
  • /ip — геолокация и ISP
  • /email — проверка утечек
  • /whois — информация о домене
  • /scan — сканирование портов
  • /hash — взлом хэша
  • /geo — детальная геолокация
  • /social — поиск по соцсетям
  • /breach — утечки паролей
  • /subdomain — перебор поддоменов

Полный список доступен по /help в интерактивном режиме. Каждая команда поддерживает экспорт результата.

Экспорт данных

Любой результат можно экспортировать в HTML с кастомными SVG‑иконками. Используйте /export /путь/папка для массового экспорта данных.

/export ./reports — создаст папку и экспортирует записи

Также после каждого поиска (телефон, IP, email) можно нажать e для сохранения отчёта.

📊 экспорты по типам последние 30 дней
phoneipemailnamewhoissocialbreach

Экспорты сохраняются в папку exports/ с временной меткой. Каждый HTML-файл включает метаданные и хэш-сумму.

Безопасность

NifyDox использует гибридное шифрование: Fernet (AES-256) для данных и PBKDF2-HMAC-SHA256 с 150 000 итераций для хранения пароля. Все записи в базе зашифрованы.

Пароль доступа к программе — Dox (можно изменить в коде, функция hash_password).

ENCRYPTION_KEY = base64.urlsafe_b64encode(hashlib.sha256(b"D0x_S3cur3_K3y_2024_M4st3r_K3y").digest())

Все экспорты содержат метку времени и хэш-сумму для верификации. Рекомендуется использовать в изолированной среде.

🔒 Совет: при работе с реальными данными всегда проверяйте источники и соблюдайте законодательство.

Адаптивность и расположение

Для ПК: боковая панель всегда видна, ширина контента ограничена 1100px для удобочитаемости. Шрифт Roboto Condensed обеспечивает отличную читаемость.

Для телефонов: при ширине экрана менее 780px боковая панель скрывается за гамбургер-меню (кнопка в левом верхнем углу). Контент растягивается на всю ширину, отступы уменьшаются.

📱 Нажмите на кнопку ☰ в левом верхнем углу (на телефоне), чтобы скрыть или показать боковую панель. На ПК кнопка отсутствует — панель всегда открыта.

Все таблицы, графики и блоки кода автоматически подстраиваются под ширину экрана. Карточки с анимацией сохраняют свою структуру даже на узких экранах.

@media (max-width: 780px) { .sidebar { transform: translateX(-100%); } .sidebar-toggle { display: flex; } }

Боковая панель имеет фиксированное положение, а основной контент прокручивается независимо. Это позволяет быстро переключаться между разделами документации.
