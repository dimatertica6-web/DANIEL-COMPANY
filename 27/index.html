SUB-DB

Консольный файловый сервер с 120 встроенными функциями. Загрузка, шифрование, бэкапы, веб-интерфейс, сканирование сети, базы данных, AI и многое другое.

pip install aiohttp colorama qrcode cryptography bcrypt pyotp dnspython whois beautifulsoup4 requests pyjwt argon2-cffi psutil
python SUB-DB.py

Установка и запуск

git clone https://github.com/sub-db/server
cd sub-db
pip install -r requirements.txt
python SUB-DB.py

После запуска откроется меню, где вы выбираете номер функции от 1 до 120. Веб-сервер запускается командой №2 на порту 8080.

SUB-DB — полностью открытый консольный файловый сервер. 120 функций, полное логирование, шифрование и безопасность.
