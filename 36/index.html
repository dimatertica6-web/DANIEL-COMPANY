Введение

ZaroDef Bot — это антивирусный Telegram-бот, разработанный с приоритетом на вашу конфиденциальность и безопасность. Мы не собираем, не храним и не передаём ваши личные данные третьим лицам.

Настоящая политика конфиденциальности описывает, как ZaroDef Bot обрабатывает информацию при использовании сервиса.

Какие данные мы обрабатываем
  • Файлы, отправленные для сканирования — обрабатываются исключительно в момент проверки и удаляются после завершения анализа.
  • URL-адреса, переданные для проверки — используются только для получения информации о безопасности ресурса.
  • ID пользователя Telegram — используется для идентификации в рамках сессии и не сохраняется в постоянном виде.
  • Хеш-суммы файлов — могут быть временно сохранены для проверки по базам сигнатур, но не содержат личной информации.

Мы НЕ собираем:

• Имена, фамилии, номера телефонов, email-адреса

• Содержимое переписки, кроме явно отправленных файлов/ссылок

• Данные о местоположении или IP-адреса (кроме явного запроса сканирования IP)

  • Для обработки команд и сканирования — файлы и ссылки анализируются локально или через защищённые API.
  • Для улучшения работы — анонимная статистика использования (количество сканирований, типы угроз) без привязки к пользователю.
  • Для безопасности — временное хранение хешей для проверки по базам вредоносных программ.
Хранение и защита данных
  • Локальное хранение: Вся информация (история сканирований, настройки) хранится локально на устройстве, где запущен бот.
  • Шифрование: Все конфиденциальные данные на диске защищены с использованием AES-256-GCM.
  • Автоматическая очистка: Временные файлы удаляются сразу после сканирования.
  • Анонимность: Режим Tor/прокси доступен для скрытия источника запросов.
Передача данных третьим лицам

ZaroDef Bot НЕ ПЕРЕДАЁТ ваши данные третьим лицам, за исключением:

  • VirusTotal API — для проверки файлов по глобальной базе сигнатур (передаётся только хеш-сумма файла).
  • Публичные DNS-серверы — при выполнении DNS-запросов (передаётся только доменное имя).

Все внешние запросы выполняются строго в рамках функционала бота и не содержат личной информации.

Ваши права
  • Доступ: Вы можете запросить информацию о том, какие данные о вас хранятся.
  • Удаление: Вы можете очистить всю историю сканирований командой /clear_history.
  • Анонимность: Вы можете использовать бота в анонимном режиме (включен по умолчанию).
  • Отказ: Вы можете прекратить использование бота в любой момент — все ваши данные будут удалены автоматически.
Изменения в политике

Мы можем обновлять данную политику конфиденциальности. О всех существенных изменениях будет объявлено в канале бота или через сообщение при запуске.

Текущая версия всегда доступна по ссылке: zarodef.com/privacy
